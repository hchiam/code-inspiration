# [Code Inspiration](https://code-inspiration.glitch.me)

Capture snippets of code inspiration while on the go.

<https://code-inspiration.glitch.me>

## Run locally

Just once:

```bash
npm install
```

and then:

```bash
npm run my-build
npm start
```
