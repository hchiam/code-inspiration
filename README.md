# Code Inspiration

Capture snippets of code inspiration while on the go.

## Run locally

```bash
yarn
```

and then

```bash
yarn start
```

## Extra info

You can generate a [dependency graph](https://github.com/hchiam/learning-dependency-cruiser) with `bash show_dep_graph.sh`.
